## Customization

### In Enhanced Version
Edit lines 18-23 in `noixzy_animate_leading_enhanced.jsx`:

```javascript
var config = {
    startLeading: 6,      // Change this
    peakLeading: 48,      // Change this
    endLeading: 6,        // Change this
    duration: 2.0,        // Change this
    easeStrength: 60      // Change this
};
```

### Values to Try
- **Fast animation:** duration = 0.5
- **Slow animation:** duration = 3.0
- **Higher peak:** peakLeading = 72
- **Tighter easing:** easeStrength = 30
- **Looser easing:** easeStrength = 100
